
# Presentación

A brief description of what this project does and who it's for

👋 Hola, soy Rubén
Desarrollador Front-End & Estratega UX/UI para Pequeños Negocios
Ayudo a negocios locales a conseguir más clientes a través de páginas web rápidas, modernas y sin rodeos. Sin laberintos técnicos ni tiendas complejas: solo diseño limpio, código eficiente y estructuras enfocadas en la conversión.

Actualmente estudio SMR y me preparo para dar el salto a DAW, combinando bases técnicas sólidas con metodologías de trabajo de vanguardia.

🚀 Mi Enfoque de Trabajo
Cero Complicaciones: Especializado en landing pages y webs presenciales con formularios directos. Tú pones el negocio, yo construyo tu escaparate digital.

Desarrollo Ágil con IA: Utilizo Inteligencia Artificial para multiplicar mi velocidad de entrega y optimizar recursos. Yo defino la arquitectura, la UX y la calidad del código; la IA es mi herramienta para agilizar el proceso x2.

Rendimiento UI/UX: Diseños 100% mobile-first pensados para que el usuario navegue y contacte sin fricciones.

💻 Stack Tecnológico
Core: HTML5, CSS3 (Flexbox/Grid)

Lógica: JavaScript (En aprendizaje continuo)

Workflow: IA Generativa (Prompting y aceleración de código), Git, Diseño UI
